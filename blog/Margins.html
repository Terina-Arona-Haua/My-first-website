Margin, Border and Padding in CSS

Margin

Margin is the space around an element, outside of its border. It is used to create space between elements. You can set margin values for each side of an element (top, right, bottom, left), or set the same margin value for all sides at once. Negative margin values are also possible, which can be used to overlap elements.

Border

Border is the line or area around an element. It is used to define the boundaries of an element. You can set the border width, style, and colour for each side of an element, or set the same border values for all sides at once. There are several border styles available, such as solid, dashed, dotted, and double.

Padding

Padding is the space between an element's content and its border. It is used to create space within an element. You can set the padding values for each side of an element (top, right, bottom, left), or set the same padding value for all sides at once.
