Margin, Border and Padding in CSS

Margin

A margin is the space outside of an element. It pushes away the elements around the chosen element.

Border

The border is the perimeter of the content which seperates the margin and the padding. It can be invisible or the style, colour and thickness can be changed.

Padding

Padding controls the space inside an element within the border. You can adjust either the size of the element or the content within the element.
